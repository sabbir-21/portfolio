A) Protein Preparation

You need to find a specific protein by literature review. Then Find the protein ID. Then follow either a) or b)

a) Protein Model from Database

if you know the protein name only, and if the PDB file is available in the Proetin data bank

1. Download Protein

You can download protein pdb file from one of the following websites.

i) RCSB

https://www.rcsb.org/
Procedure: go to the website > search the protein > click download files > download PDB format
Or,

ii) Swiss Model

Procedure: go to the website > search the protein > click download files > download PDB format
Or,

iii) PDBE

Or,

iv) wwpdb

https://www.wwpdb.org/
Procedure: go to the website > search the protein > click download files > download PDB format
Or,

2. Minimize Protein

After downloading the protein PDB file, you need to minimize the energy of the protein.

i) Swiss PDB Viewer

Procedure: open spdbviewer > click file > open the protein model PDB file > click select > click all > click tools > click energy minimisation > again click file > save > save current layer > file rename as energy minimized

ii) Discovery Studio

Procedure: Open Discovery studio > open PDB file > click scripts > Publication quality > click selection > remove water groups, Hetatm groups, Ligand groups > save as energy minimized PDB file

iii) Pymol

Procedure: Open Pymol > Load PDB file > click selection > remove water groups, Hetatm groups, Ligand groups > save as minimized PDB file

B) Drug Preparation

You need to find a control drug by literature review. Then you have to search similar drugs or chemicals of it. Follow the steps bellow:

a) Drug Similarity Searching

Use any website given bellow. You can use multiple website for better similarities search. I used PubChem+Ambinter+Swiss Similarity. And for all chemical compounds in a specific plants, I use IMPPAT

1. Pubchem

https://pubchem.ncbi.nlm.nih.gov/
Procedure: go to the website > search Control drug > download control drug 3D conformer sdf file > sdf file open with Notepad > copy the total sequence from notepad

2. Ambinter

https://www.ambinter.com/#search
Procedure: go to the website > Paste the notepad sequence > different results showed > collect different smiles id

3. Swiss Similarity

http://www.swisssimilarity.ch/
Procedure: go to the website > paste control drug smiles id > select a class of compound (Drugs) > choose DrugBank > choose different Pathway > submit > download CSV file > collect smiles id which has more than 0.70 similarities. Then, choose ChEMBL-approved drugs > choose a different Pathway > submit > download CSV file > collect smiles id with more than 0.70 similarities. Then, ChEMBL drug candidates in clinics > choose a different Pathway > submit > download CSV file > collect smiles id with more than 0.70 similarities.
To download all chemical by one click, I developed a software. Try it

b) Combine Drugs

You have to combine or merge all drugs in a one file. Use OpenBabel to do this.

1. Open Babel

Procedure: Open OpenBabel > choose “sdf - MDL MOL format” as an input format > choose “sdf - MDL MOL format” as an output format > choose output file location > Input all sdf file > click convert

C) Active Site Identification

To identify Active Site open pdb file in discovery studio > Click Active sites > Right click > Labels > Add > Amino Acid > Right click > Display Style > CPK > Apply > OK

D) Molecular Docking

You can use one of the following software. But we will use PyRx for easy use.

a) PyRx

You have to do next three steps.

1. Protein Input

Procedure: File > Load Molecule > right click molecule > AutoDock > Make Macromolecule

2. Drugs Input

Procedure: File > Load Molecule > right click molecule > AutoDock > Make Micro molecule
Or, Below the PyRx, you’ll see Controls (Vina Wizard, Autodock Wizard, OpenBabel etc) Procedure: Click OpenBabel > Click insert a new item (On bottom right corner) > Select All ligands merged by Openbabel [see step in B>b>1] > Right click of any row > Minimize All > Again right click > Convert All to Autodock Ligand (pdbqt)

3. Start Docking

Procedure: Go to Vina Wizard > Start > Select All ligands(pdbqt) [shortcut for select all: Click first one, then Shift+Last one All will be selected] > select Macromolecules > click Forward > Click on maximize > Forward > Docking will start > after finishing of docking > Save results as CSV file from bottom right corner > Also export file from top menu bar (as .tar.gz)

E) ADMET Analysis

You need ADMET data analysis of the chemical that scores best in the docking. Use both website bellow to analysis

a) Swiss ADMET

http://www.swissadme.ch/
You need SMILES id of the drugs that results best in the docking score. Enter SMILES and save the response in an excel file.

b) PKCSM

https://biosig.lab.uq.edu.au/pkcsm/prediction

c) ADMET LAB 2.0

https://admetmesh.scbdd.com/
You need SMILES id of the drugs that results best in the docking score. Enter SMILES and save the response in an excel file.

F) Figure Preparation

You need ADMET data analysis of the chemical that scores best in the docking. Use both website bellow to analysis

a) 2D+3D image

Open both drug and protein file in Discovery Studio. Then Scripts > Visualization > Publication Quality.
Select Energy minimized protein and define as receptor.
Select Drug and define as Ligand. Then Click Ligand interaction. Then click show 2D diagram.
Right click>Display Style>Change font (Times New Roman) and size 1.5>Click Residue> Disc> Click Interaction>Show distance
Take a screenshot.
Simultaneously After generating 2d image, click Receptor. Select Interacting Receptor Atom>Right click>Label>Add>Select Amino Acid from dropdown>Change font (Times New Roman), Size 12, Color Black>Apply>OK
Rotate to better view of all amino acids>Delete green circle carefully>Take a suitable screenshot.
Combine 2D and 3D image in png format in an image editor like Paint/Illustrator/Photoshop. I used illustrator. The image look like this.
